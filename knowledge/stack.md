# Technology Stack

This document outlines the technical stack and tools used by Ajay Susanth across backend development, cloud engineering, DevOps, frontend, and AI integrations.

---

## Languages
*   **Python**: Primary language for backend services, machine learning integrations, and scripting.
*   **JavaScript**: Used for frontend development, Node.js applications, and custom automation.
*   **TypeScript**: Used to ensure type-safe React applications and frontend components.

---

## Backend
*   **FastAPI**: Highly performant, async Python framework used to build RESTful APIs.
*   **Express.js**: Minimalist Node.js framework utilized for lightweight routing and automation services.

---

## Frontend
*   **React**: Core framework for building dynamic, single-page user interfaces.
*   **Tailwind CSS**: Utility-first CSS framework for rapid UI styling and design systems.

---

## Databases
*   **PostgreSQL**: Relational database for persistent application data.
*   **pgvector**: PostgreSQL extension for high-performance vector similarity searches.
*   **MongoDB**: Document-based NoSQL database used for logging and unstructured state storage.
*   **Redis**: Key-value data store used for API rate limiting, response caching, and worker task queuing.

---

## Cloud
*   **Microsoft Azure**: Core public cloud platform.
*   **Azure Container Apps**: Fully managed serverless container service.
*   **Azure Container Registry**: Secure hosting for Docker images.
*   **Azure PostgreSQL Flexible Server**: Managed relational database service.

---

## DevOps
*   **Docker**: Containerization engine for application portability and environment consistency.
*   **Kubernetes**: Container orchestration platform for managing microservices.
*   **Azure Kubernetes Service (AKS)**: Managed enterprise Kubernetes cluster.
*   **Helm**: Package manager for deploying and managing Kubernetes manifests.
*   **GitHub Actions**: CI/CD tool to automate testing, building, and deployment runs.
*   **ArgoCD**: Declarative GitOps continuous delivery tool for Kubernetes.
*   **Terraform**: Infrastructure as Code (IaC) tool for provisioning cloud infrastructure.
*   **Calico Network Policies**: Custom security policies defining network isolation between pods.
*   **OPA Gatekeeper**: Policy controller enforcing governance and compliance in Kubernetes.
*   **Secrets Store CSI Driver**: Mounting secrets from Key Vault into Kubernetes pods.
*   **Prometheus**: Open-source systems monitoring and alerting toolkit.
*   **Grafana**: Analytics and visualization platform for monitoring dashboards.
*   **KEDA**: Kubernetes Event-driven Autoscaling to scale workloads based on metrics.

---

## AI / ML
*   **Embeddings**: Text vectorization techniques (`text-embedding-3-small` model).
*   **Vector Search**: Semantic similarity matching using cosine similarity metric.
*   **LLM Integration**: Working with API endpoints like Google Gemini and OpenAI.
*   **Natural Language Processing (NLP)**: Text processing, context mapping, and classification.
*   **Document Intelligence**: Azure Document Intelligence for parsing scanned files and PDFs.
*   **Named Entity Recognition (NER)**: Extracting structured entities (e.g. skills, names) from unstructured texts.

---

## Tools
*   **n8n**: Workflow automation tool for integrating external APIs.
*   **Telegram Bot API**: Building interactive communication endpoints.
*   **Prisma ORM**: Object-Relational Mapping library for database access.
*   **Postman**: Tool for designing, testing, and documenting APIs.
